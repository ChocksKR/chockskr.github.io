Two Sum

Description

Given an array of integers, return indices of the two numbers such that they add up to a specific target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

Example:

Given nums = [2, 7, 11, 15], target = 9,

Because nums[0] + nums[1] = 2 + 7 = 9,
return [0, 1].

Algorithm

1. Iterate through the elements in the list
2. Find the difference between the target and element
3. If that element is present in dictionary then we have obtained the solution so return the element and diff value
4. else insert that element into the dictionary

Brute Force Solution

The first way you could go about solving this challenge is likely the way that most people come up with when they first see the problem.

The issue is that you have a worst case run time of O(n²). If the two numbers you are searching for are at the end of the list, you would need to go through all of the numbers multiple times, once in the “X” loop and each respective time in the “Y” loop.
This would give you the correct answer but the runtime is slow, and in the case of a large number of numbers it could be very problematic.
